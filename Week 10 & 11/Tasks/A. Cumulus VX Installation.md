- # Cumulus VX Installation
  - Installing Cumulus VX:
    - Go to this link : https://www.nvidia.com/en-us/networking/ethernet-switching/cumulus-vx/download/
    - Login and download the VirtualBox one. (.ova) 
  - Accessing Cumulus VX:
    - Import previously download .ova image in VirtualBox.
    - Login into the cumulus VX by following username and password:
      ```
      Username: cumulus
      Password: cumulus
      ```
      - It will prompt you to change the password.
